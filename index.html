¿Qué es MineDex?

MineDex es una enciclopedia digital de código abierto diseñada para aventureros. Nuestra misión es catalogar cada **bloque**, cada **criatura** y cada rincón de este universo infinito.

Ya seas un arquitecto buscando el ID de un material específico o un guerrero estudiando los puntos de vida de un jefe, aquí encontrarás la información técnica y visual que necesitas para sobrevivir y prosperar.
